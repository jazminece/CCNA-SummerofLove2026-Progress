Assessment Overview

This activity involved exploring and configuring a simulated network environment, focusing on basic networking tools, device identification, and router configuration.

Parts Completed

Part 1: Investigate the Bottom Toolbar
Explored the simulator interface and learned how to use navigation tools such as zoom, select, and interaction controls.

Part 2: Investigate Devices in a Wiring Closet
Identified and examined networking equipment inside the wiring closet, including switches and related infrastructure components.

Part 3: Connect End Devices to Networking Devices
Practiced connecting end devices (such as PCs) to network devices using appropriate cabling and port selection.

Part 4: Install a Backup Router
Added and positioned a backup router within the network topology to support redundancy and connectivity.

Part 5: Configure a Hostname
Accessed the router via terminal and configured its hostname using basic CLI commands.

Part 6: Explore the Rest of the Network
Navigated through the remaining network topology to understand device relationships and overall structure.

Key Learnings
Basic navigation and interaction within a network simulator
Identification of common networking devices and their roles
Physical and logical connection of network components
Basic router configuration using CLI (including hostname setup)
Improved familiarity with zooming, inspecting, and exploring network layouts
